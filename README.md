# Simple fight game with monster

## Use

* HTML5
* CSS3
* VUE
* WebPack


### Installing

clone repository

```
git clone https://github.com/jakublack/game-fight-vue.git
```

install project dependencies

```
npm install
```

### Development

Start project in dev mode

```
npm run serve
```

### Production
Build production pack

```
npm run build
```
