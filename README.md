# Simple game fight with Monster

## Use

* HTML5
* CSS3
* VUE
* WebPack

Run on Netlify:
```
https://upbeat-wright-69d548.netlify.com
```

### Installing

clone repository

```
git clone https://github.com/jakublack/game-fight-vue.git
```

install project dependencies

```
npm install
```

### Development

Start project in dev mode

```
npm run serve
```

### Production
Build production pack

```
npm run build
```
